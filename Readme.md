# 🚗 AutoMarket - Marketplace Automotivo

Plataforma digital nacional de compra e venda de veículos usados e seminovos conectando compradores, vendedores particulares e lojas.

## 🛠️ Tecnologias Utilizadas
- **Front-end:** React, TypeScript, Tailwind CSS (via Lovable)
- **Back-end/Banco de Dados:** Supabase (PostgreSQL)

## 🎯 Escopo do Desafio
- Busca avançada com múltiplos filtros combináveis.
- Comparador lado a lado de até 3 veículos simultâneos.
- Módulo de anúncios persistido de forma relacional, mitigando alucinações de mídia.
